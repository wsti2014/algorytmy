**Zad. 8**: Znaleźć wartość minimalną w ciągu liczb dodatnich zakończonym wczytaniem 0.

Dane: *p*
	
```c++
wczytaj x; // ciąg jest czytany do końca
if (x == 0)
	wyprowadź -1; // ciąg jest pusty
else {
	min = x
	while (x > 0) {
		if (x < min)
			min = x;
		wczytaj x;
	}
}
wyprowadź min; // liczba najmniejsza
```
